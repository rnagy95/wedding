# Wedding invitation website

This project is a minimal, elegant Jekyll site for the wedding invitation.

## Structure

- [index.md](index.md) – main content in Markdown
- [_layouts/default.html](_layouts/default.html) – shared page shell
- [assets/css/style.css](assets/css/style.css) – styling for the pastel, minimal design
- [.github/workflows/jekyll.yml](.github/workflows/jekyll.yml) – GitHub Pages build and deploy workflow

## Local development

1. Install Ruby and Bundler.
2. Run `bundle install`.
3. Run `bundle exec jekyll serve`.

The site is designed to be hosted on GitHub Pages.
